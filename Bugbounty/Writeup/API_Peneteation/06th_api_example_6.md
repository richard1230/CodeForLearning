## 复现步骤

1.进入你的mopub帐户，在你的网络中创建一个网段。


2.你现在会得到一个segment ID。


3.现在转到API链接:`https://app.mopub.com/networks/v2/api/segment/%5BSegment_id%5D`  <br>

注意:页面将花费很多时间打开，你的浏览器可能会崩溃,因为响应包含了mohub中有hash键的所有Apps。

4.当页面打开时，你可以在App部分看到所有的App。
